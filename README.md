 # CineVerse

This is my WAP project.

I made a movie website using HTML CSS and JavaScript. It shows popular movies using an API. I also added search filter and sort features so users can easily find movies.

The movies are coming from TMDB API and are displayed dynamically using JavaScript.

Tech used in this project are HTML CSS and JavaScript.

To run this project just open index.html in browser.

In this project I learned how to use fetch API DOM manipulation and how to implement search filter and sorting.
